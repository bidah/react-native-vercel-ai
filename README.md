# react-native-vercel-ai

run vercel ai package on react native, expo, web and universal apps.

## Installation

```sh
npm install react-native-vercel-ai
```

## Usage

```js
import { multiply } from 'react-native-vercel-ai';

// ...

const result = await multiply(3, 7);
```
